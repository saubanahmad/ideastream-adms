# Download httplib.h

## Option 1: Direct Download (Recommended)
Visit this URL and save as `httplib.h`:
https://raw.githubusercontent.com/yhirose/cpp-httplib/master/httplib.h

## Option 2: PowerShell Download (Windows)
Run this command in PowerShell:
```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/yhirose/cpp-httplib/master/httplib.h" -OutFile "httplib.h"
```

## Option 3: curl Download (Linux/Mac/Windows Git Bash)
```bash
curl -o httplib.h https://raw.githubusercontent.com/yhirose/cpp-httplib/master/httplib.h
```

## Option 4: wget Download (Linux)
```bash
wget https://raw.githubusercontent.com/yhirose/cpp-httplib/master/httplib.h
```

## Option 5: Manual Download from GitHub
1. Go to: https://github.com/yhirose/cpp-httplib
2. Click on "httplib.h" file
3. Click "Raw" button
4. Save the page as `httplib.h`

## Verify Download
The file should be:
- Name: `httplib.h`
- Size: ~600-800 KB
- Location: Same folder as `SocialConnectWeb.cpp`

## After Download
Run the setup script:
- Windows: `setup.bat`
- Linux/Mac: `./setup.sh`
