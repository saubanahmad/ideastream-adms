# SVG Icons

This folder is for SVG icons used in the frontend UI.

Icons should be named clearly, like:

* `home.svg`
* `profile.svg`
* `search.svg`
* `notification.svg`
* `like.svg`
* `comment.svg`
* `share.svg`
* `friends.svg`
* `community.svg`
* `logout.svg`

Prefer simple lowercase kebab-case names if needed, like `create-post.svg`.
